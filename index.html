    Projects and Publications

    Shooter? I Hardly Know Her

    Shooter? I Hardly Know Her is a Unity3D online multiplayer shooter built in C# for my capstone. I developed core gameplay systems, including player movement, enemy AI, and a weapon spawner system, optimizing performance for smooth online play. The game has undergone 30+ playtests (avg. 8.3/10) and is planned for Steam release.

    Conversational Procedural Content Generation with LLMs

    A research project exploring LLM-driven procedural content generation through conversational interactions. I co-authored a workshop paper (FDG 2025, PCG Workshop, ACM pending) and developed two prototypes in Phaser to test LLM-based world generation. Additionally, I collaboratively engineered TinyTownQA, a dataset with 31 procedurally generated maps and 642 spatial reasoning questions, benchmarking LLM accuracy at up to 89.3%.

    Automanora

    A Unity3D Puzzle Platformer. Features multiple custom built puzzle levels, inventory system and dynamic battery powered mechanisms. Won the Best Aesthetic award (voted by peers) in CMPM 125: Game Technologies at UCSC.

    Phaser Roguelike

    A roguelike game built using Phaser.js. Features a custom built level, weapon upgrade mechanics and a boss fight.

    Phaser Gallery Shooter

    A fast-paced gallery shooter game developed with Phaser.js. Includes multiple enemy types, waves, and power-ups for an engaging gameplay experience.

    ZDOC

    A Survival Unity2D Game built with C#. Player type different patterns using arrow keys to "heal zombies" Features multiple enemy types; custom built map, assets and animations; and multiple power ups.

    Machine Learning Project

    This project explores the relationship between self-esteem and academic performance. Using machine learning techniques in R like Lasso, Ridge, and Random Forest, we analyzed survey data to predict academic outcomes.

    Livance

    A health social media app designed for a hackathon to help users share health data, and seek resources with friends and families. Built with a focus on user-friendly design and efficient task tracking.

    Algorithmic Trading

    A learning-focused project exploring algorithmic trading strategies using Python. Includes backtesting, data analysis, and visualization of trading performance using a sandbox API.
